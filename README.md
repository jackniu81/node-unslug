# node-unslug
un-slug in node.js implementation. Change slug 'hello-world' to 'Hello World'

## Usage

	var unslug = require('./');
	
	console.log(unslug('hello-world'));	
	// outputs: Hello World
	
	console.log(unslug('a-simple-demo'));
	// outputs: A Simple Demo



## unslug on NPM
You can install node-unslug via NPM, like so:

	npm install unslug
